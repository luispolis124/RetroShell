# RetroShell
RetroShell: Uma interface unificada e imersiva para emulação multi-console via web. Atua como um "shell" moderno para engines de retro-gaming, oferecendo suporte a múltiplos núcleos, controles touch responsivos e periféricos. Transforme seu navegador em um centro de jogos clássicos completo, focado em performance e fidelidade visual.
